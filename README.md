# 🍕 Pizzaria Web

Sistema web completo para pizzarias — cardápio online, pedidos, painel administrativo e autenticação.

## Tecnologias
- **Backend:** FastAPI (Python)
- **Frontend:** React + Vite
- **Banco:** SQLite + SQLAlchemy
- **Auth:** JWT com refresh token

## Funcionalidades
- Cardápio com tamanhos, sabores e adicionais
- Pedidos online com endereço e forma de pagamento
- Painel admin (produtos, preços, imagens)
- Autenticação completa (cliente / administrador)
